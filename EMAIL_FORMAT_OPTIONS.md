# Email Format Options

## Current Format
- Fancy HTML template with styling, buttons, etc.

## Option 1: Simple Plain Text (Recommended)
Just the data in a clean text format:
```
CONSULTATION REQUEST
===================
Name: John Doe
Email: john@example.com
Phone: 123-456-7890
Age: 35
Consultation Type: Hair Transplant
Preferred Date: January 15, 2025
Date Submitted: Jan 10, 2025 2:30 PM
```

## Option 2: Simple HTML Table (Minimal Styling)
Clean table with basic styling - easy to read in email clients.

## Option 3: CSV Format
Data in comma-separated format that can be imported into Excel/Google Sheets.

## Option 4: JSON Format
Structured data format for easy parsing.

## Option 5: Simple HTML with Just Data
Minimal HTML, just the data in a readable format.
